# Dissemination Packages

This directory contains platform-ready packages for wider reuse and validation.

## Recommended Public Asset

Publish this as a **dataset/research asset**, not a model. The project produced evidence tables, pathway-intervention mappings, omics outputs, enrichment outputs, figures, scripts, and manuscript materials. It did not produce a trained clinical prediction model.

## Packages

- `kaggle_pd_evidence_dataset/`: Kaggle dataset package with `dataset-metadata.json`.
- `hf_pd_evidence_dataset/`: Hugging Face dataset repository package with dataset-card `README.md`.
- `upload_to_huggingface.py`: helper upload script using `huggingface_hub`.

## Recommended Tags

`parkinsons-disease`, `neuroscience`, `bioinformatics`, `transcriptomics`, `clinical-trials`, `drug-repurposing`, `public-health`, `network-analysis`, `evidence-synthesis`

## Caution

This dataset is for research, education, reproducibility, and hypothesis generation. It is not a diagnostic tool, clinical recommendation system, or proof that any intervention prevents or cures Parkinson's disease.
